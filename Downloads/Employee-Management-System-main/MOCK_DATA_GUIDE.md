## üÜò Troubleshooting

### If data is not showing:
1. Make sure MongoDB is running
2. Check backend server is running on port 5000
3. Check frontend server is running on port 5173
4. Clear browser cache and reload
5. Re-run the seed script

### To verify data in MongoDB:
```bash
# Connect to MongoDB
mongosh mongodb://localhost:27017/ems

# Check collections
db.users.countDocuments()
db.projects.countDocuments()
db.tasks.countDocuments()
db.attendances.countDocuments()
```
